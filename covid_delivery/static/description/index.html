dhruv_2_6_22

Video Tutorial

dhruv_2_6_22 module allows you to create records in draft, done and cancel state. You will also create record in the Contacts module from contact_cration web form.

  • Can create record in the form view in any of the stage on button click which is visible through odoo workflow.
  • After that on click of Proceed Operation date_order will be set of operastion date.
  • Also create a record in res_partner through contact_creation web form.
  • And by submitting the form, record is created in Contacts.

Prerequisites before Installation

Certain Prerequisites are required before the installation.

  • Install Sale module from Apps.
  • Install Contacts module from Apps.
  • Install Website module from Apps.
